# MVP Scope — Fitness AI App

## Core Features

### 1. Authentication
- Register
- Login
- JWT auth

### 2. Activity Tracking
- Start / Stop activity
- Track:
  - duration
  - distance
  - pace

### 3. GPS & Map
- Real-time tracking
- Route display (polyline)

### 4. Activity History
- List past activities
- View activity details

### 5. Dashboard
- Weekly stats
- Monthly stats
- Total distance

### 6. Goals
- Set weekly distance goal
- Track progress

### 7. AI Features
- Weekly summary
- Simple AI chat (fitness insights)

---

## Excluded (Post-MVP)
- Social features
- Wearables
- Meal tracking
- Voice assistant
